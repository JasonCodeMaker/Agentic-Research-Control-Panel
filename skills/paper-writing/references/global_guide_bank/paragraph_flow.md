# Guide: Paragraph Flow (reverse outline)

Use whenever asking whether a paragraph or section "flows".

## One-paragraph clarity test
1. Does the paragraph have **one** explicit message?
2. Does the **first sentence** state what the paragraph will do?
3. Are all key nouns/terms readable without hidden context?
4. Does each sentence connect to the previous one with a clear relation — cause, contrast, consequence,
   refinement, or example?

## Reverse outlining (per section)
- Write the section thesis.
- Write each paragraph's topic sentence.
- Write the evidence/explanation points under each paragraph.
- Check the mapping: topic sentence → thesis, evidence → topic sentence.
- Revise or remove any paragraph that cannot be mapped cleanly.

## Flow audit (between paragraphs)
- Read the last sentence of paragraph N and the first sentence of N+1. Does the transition work?
- Composition — the logical flow a reader can follow — is the most important property of technical writing.

## Empty-transition smell
Openers like "Furthermore,", "Moreover,", "Additionally," used as glue between unrelated points signal a
missing logical relation. Replace with the actual relation or merge the paragraphs.
